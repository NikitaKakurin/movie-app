Привет, работу оцениваю в 60 баллов. Работа поддерживает разрешение экрана от 320px и выше. Реализована сортировка по году, жанру, а также популяпрности, дате и рейтингу. Сортировку можно комбинировать. Также реализован поиск и пагинация. В случае отсутствия постера вставляется картинка "No poster". Также в случае ошибок в ответах Api появляется alert c кодом ошибки. При клике на карточку фильма - рендерится окно с кратким описанием фильма, фоном, и трейлером. В предоставленных API видео сначала происходит поиск Тизера/Трейлера, а если их нет, вставляется первое в списке видео.
